Detailed Description

Small block size summary added by the AliHLTComponent base class if component statistics are enabled (–enable-compstat).

fLevel is retrieved from incoming block statistics and incremented. Incoming block statistics are appended to the newly added one if –enable-compstat=full has been chosen.

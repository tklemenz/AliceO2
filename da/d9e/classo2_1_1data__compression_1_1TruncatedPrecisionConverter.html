Detailed Description

template<class ParameterModelT>
class o2::data_compression::TruncatedPrecisionConverter< ParameterModelT >

A simple converter producing truncated precision The converter implements the write function needed to be used as a codec in the data compression framework. Simply a prototype case for the moment.

The parameter model is required to implement the method 'convert'.

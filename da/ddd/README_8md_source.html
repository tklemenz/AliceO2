2 ALICE O2 software
3 =================
4 
5 ### Scope
6 The ALICE O2 software repository contains the framework, as well as the detector specific, code for the reconstruction, calibration and simulation for the ALICE experiment at CERN for Run 3 and 4. It also encompasses the commonalities such as the data format, and the global algorithms like the global tracking.
7 Other repositories in AliceO2Group contain a number of large common modules, for instance for Monitoring or Configuration.
8 
9 ### Website
10 The main entry point for O2 information is [here](http://alice-o2.web.cern.ch/).
11 A quickstart page can be found under [https://aliceo2group.github.io/](https://aliceo2group.github.io/).

37 ### Formatting
38 ##### Using clang-format
39 
40 ~~~~~~~{.sh}
41 
42 # Get the configuration file
43 git clone https://github.com/AliceO2Group/CodingGuidelines.git
44 cp CodingGuidelines/_clang-format-4 /path/to/O2/top/dir # (use _clang-format-3 if you use clang-format v3)
45 
46 # Check the style
47 # Here any tag "<replacement " indicates a problem ("<replacements " with **s** is fine!)
48 clang-format -style=file -output-replacements-xml SOURCEFILE`
49 # shows what would the file content be after the reformatting
50 clang-format -style=file SOURCEFILE
51 
52 # Apply the style to the file
53 clang-format -style=file -i SOURCEFILE
54 ~~~~~~~

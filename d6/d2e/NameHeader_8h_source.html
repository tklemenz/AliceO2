o2::header::NameHeader
an example data header containing a name of an object as a null terminated char arr. this is a template! at instantiation the template parameter determines the size of the held string array. a caveat with decoding is you have to use header::get<NameHeader<0>>(buffer) to get it out of a buffer. May improve in the future if enough people complain.
